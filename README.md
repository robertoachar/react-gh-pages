# react-gh-pages

> React on GitHub Pages.

# Development

* Clone the repo

```bash
$ git clone https://github.com/robertoachar/react-gh-pages.git
```

* Install dependencies

```bash
$ npm install
```

* Run project

```bash
$ npm start
```

* Deploy app

```bash
$ npm run deploy
```
